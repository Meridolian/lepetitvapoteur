# Le Petit Vapoteur Mobile !
#
# Ce projet est un prototype d'application pour smartphone du e-commerce Le Petit Vapoteur, leader français du marché de la cigarette électronique.
# 
# Il propose les mêmes fonctionnalités que le site web actuel, tout en ajoutant une meilleure expérience utilisateur, pour les utilisateurs de smartphones.
#
# Il a été réalisé grâce à Nativescript Angular, un framework Javascript permettant de faire des applis mobile Cross-Platform, c'est-à-dire disponible pour Android et IOS.
